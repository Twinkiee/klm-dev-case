Destination Finder Resource definition

  • Base URI: http://www.klm.com/destinations/destination-finder/
  • Query parameters: origin, pos, maxBudget, minBudget
Query parameter description
  • Origin: The 3 letter origin airport IATA code, i.e. AMS
  • POS: Point of sale, i.e. NL
  • maxBudget: The maximum budget allowed for the fare.
  • minBudget: The minimum budget allowed for the fare.
